# Vera Recall (OpenCode Fallback Instruction)

Purpose: Provide useful memory recall that answers the question before listing evidence.

Requirements:
- Infer intent from loose phrasing (status, timeline, decision, or general recall).
- Start with the best answer first, then add concise supporting evidence.
- Keep file provenance visible in evidence lines.
- Be explicit about missing data and unavailable files.
- Preserve raw mode for integrations with complete metadata.
